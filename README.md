# esm-structure-eval
Benchmarking ESMFold2 (may 2026) predicted structures against experimental PDB data.

This is a model built off of ESMC which is a protein language model that's trained on 2.8 billion sequences.

Spearheaded by Alex Rives the head of science at Biohub and the former Meta protein team.
Alex Rives is interesting to me because he believes in the scaling laws much like I do. Think 'The Bitter Lesson' by Richard Sutton.
